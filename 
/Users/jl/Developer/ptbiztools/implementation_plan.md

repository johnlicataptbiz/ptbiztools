# Implementation Plan

## Overview
Refactor the Danny tools (DannyCloserCallGrader.tsx, DannyCompensationCalculator.jsx, DannyFinancialAudit.jsx) to production standards based on previous reviews: split large components, add TypeScript, extract styles, fix missing imports, add tests, improve maintainability while preserving all functionality.

Context: Next.js 16/React 19 monorepo frontend with Tailwind, Vitest testing setup added. Three tools share design system but have individual issues (monster components, inline styles, @ts-nocheck, no tests). Goal: Consistent, typed, testable components ready for team development.

High-level: Extract shared theme/CSS, convert to TSX with types, split into subcomponents, add Vitest tests (80% coverage), fix bugs (missing imports), preserve UX/print features.
