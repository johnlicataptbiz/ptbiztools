# Project UI Enhancer

## Instructions
Enter full intelligent UI + assets upgrade mode:
1. Auto-scan and classify every image/SVG/graphic in the workspace.
2. Auto-rename ugly files to clean semantic names (example: "Generated Image March 13, 2026 - 11_02AM.png" → "dashboard-hero-background.png").
3. Auto-move and organize into logical folders inside public/assets/.
4. Deep-analyze the entire Next.js component tree and auto-suggest 8-10 perfect placements.
5. Present a clean table showing before/after filenames, new locations, and visual reasoning.
6. Wait for user approval (numbers or "all").
7. For approved items: auto-apply renames + moves + modern Tailwind/React code (glassmorphism, gradients, hover effects, responsive).
8. Commit everything safely, run build + lint validation.
9. Update PROJECT_CONTEXT.md with new "UI Enhancements" section including before/after health score.

Safety: Always show full diff preview of renames + code changes, keep originals as backup until approved, auto-rollback on any failure.

Basic usage: User says "Enhance UI with custom graphics" or "Make the site beautiful with my assets."
Automation triggers: Auto-runs after project-dependency-enhancer or when user mentions "ui upgrade", "rename graphics", "organize assets", "make it look better".
Chaining and automation: Runs automatically after full rescue + dependency enhancer, then triggers post-rescue-validation.
Advanced automation: User says "Run full rescue and enhance UI beautifully" — executes the entire chain end-to-end with renaming, organizing, and integration without extra prompts.
